# Image Resizer
Image resizing applicationg based on Seam carving algorithm

### Options

  - "GPU" - use GPU for calculations
  - "Autosave" - saves result image in current directory as "result.png"

### Dependencies

  - Qt 5.2
  - OpenCl 2.2
